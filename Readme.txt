# Portfolio Website - Civil Engineer 

A customized portfolio website originally based on a template, modified to showcase a civil engineer's projects and professional skills. Added new sections and personalized design elements to make the portfolio more engaging.

---

##  Project Overview
This project demonstrates how to adapt a generic template to a professional portfolio, highlighting projects, skills, and achievements.  
The modifications include additional sections and custom styling to better represent the engineer's profile.

---

##  Tech Stack
- HTML5
- CSS3
- JavaScript
- Responsive design
- Template customization

---

##  Features
- Added new sections to the original template
- Display of civil engineering projects
- Skills and experience showcase
- Clean and modern UI
- Fully responsive across devices
